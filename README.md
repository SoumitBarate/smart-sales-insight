# 📊 Smart Sales Insight Generator

An AI-powered data analytics tool that automatically generates business insights from sales data.

## 🚀 Features
- Upload CSV dataset
- Automatic KPI generation
- AI-generated insights using LLM
- PDF report download
- Interactive Streamlit dashboard

## 🛠 Tech Stack
- Python (Pandas)
- Streamlit
- OpenRouter API (LLM)
- ReportLab

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
